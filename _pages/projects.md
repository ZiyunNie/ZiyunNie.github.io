## 📃 Projects
### Current Projects

### 🧠 Adaptive Learning in Autism (Computational Modeling)

This project investigates how autistic children update beliefs and accumulate evidence in dynamic environments using a probabilistic reversal learning paradigm.
 
- Applied reinforcement learning and drift diffusion models (EZ-DDM) to explore the process of probabilistic reversal reward learning
- Identified reduced evidence accumulation and atypical belief updating in autistic children  

---

### 👁️ Social Attention and Diagnosis (Eye-tracking + Machine Learning)

This project aims to improve the detection of ASD-related social attention patterns using machine learning applied to eye-tracking data.

- Led data preprocessing and feature engineering for eye-tracking dataset (N = 148)  
- Implemented supervised learning models with cross-validation  
- Conducted model interpretability analyses (feature importance)  

---

### 🤝 Understanding Irony in Autistic Children (Eye-tracking + fNIRS Hyperscanning)

This project examines single-person eye movements and brain-to-brain synchrony during adult–child interaction in irony comprehension.

- Designed and conducted eye-tracking and fNIRS hyperscanning experiment (adult–child dyads)  
- Investigating neural synchrony of social communication and interaction

---

### Previous Work

### 😐 Emotion Processing and Attentional Bias in High-AQ Individuals

This project examined how individuals with higher autistic traits process emotional facial expressions.

- Designed and conducted eye-tracking experiments  
- Applied random forest and Bayesian modeling (PyMC)  
- Identified atypical attention patterns and faster evidence accumulation under ambiguous emotional conditions  

---

### 📊 Social Attention and Autistic Traits: A Meta-analysis

This project synthesized findings on gaze cueing across the autism trait continuum.

- Screened ~3,500 studies and coded ~300 eligible papers  
- Conducted meta-analysis using R (metafor)  
- Evaluated effect sizes and publication bias
